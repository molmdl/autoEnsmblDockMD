# autoEnsmblDockMD
Scripts and experimental agent skill to automate a workflow of ensemble docking + MMPBSA

## What is autoEnsmblDockMD
* Just a toy project of vibe coding. 
* Aiming to convert a manual routine workflow into an automated, agent-friendly workflow.
* Starting with a set of (unorganized, inconsistent) human-written bash scripts, LLM-generared python scripts, and human-written guidelines.
