```python
def detect_execution_backend() -> str:
    """Detect execution environment (local vs slurm). Returns 'local' or 'slurm'."""

class LocalExecutor:
    """Local command execution."""
    
    def __init__(self):
        """Initialize local executor."""
        
    def run_command(self, command: List[str], cwd: str = None, 
                    env: Dict[str, str] = None, timeout: int = None) -> Tuple[int, str, str]:
        """Run command locally, return (returncode, stdout, stderr)."""
        
    def run_script(self, script_path: str, args: List[str] = None, **kwargs) -> Tuple[int, str, str]:
        """Run a script file."""

class SlurmExecutor:
    """Slurm job submission and management."""
    
    def __init__(self, account: str = None, default_time: str = '02:00:00',
                 default_cpus: int = 1, default_mem: str = '4G'):
        """Initialize Slurm executor with defaults."""
        
    def submit_job(self, script_path: str, job_name: str = 'autoEnsmbl',
                   time: str = None, cpus: int = None, mem: str = None,
                   output: str = None, error: str = None,
                   extra_args: List[str] = None) -> str:
        """Submit job to Slurm, return job ID."""
        
    def get_job_status(self, job_id: str) -> str:
        """Get job status: PENDING, RUNNING, COMPLETED, FAILED, CANCELLED, TIMEOUT, UNKNOWN."""
        
    def wait_for_job(self, job_id: str, poll_interval: int = 30, 
                     timeout: int = None) -> Tuple[str, str]:
        """Wait for job completion, return (status, output_file)."""
        
    def cancel_job(self, job_id: str) -> bool:
        """Cancel a job."""
        
    def get_queue_info(self, user: str = None) -> List[Dict[str, str]]:
        """Get queue information for user."""
```

Implementation requirements:

1. **detect_execution_backend():**
   - Check SLURM_JOB_ID, SLURM_TMPDIR, SLURM_NODELIST env vars
   - Return 'slurm' if any SLURM_* var is set, else 'local'

2. **LocalExecutor:**
   - Use subprocess.run with capture_output=True
   - Handle timeout with proper cleanup
   - Return tuple of (returncode, stdout, stderr)
   - Support environment variable override

3. **SlurmExecutor:**
   - Use subprocess.run to call sbatch/squeue/scancel
   - Parse job ID from sbatch output: "Submitted batch job 12345" → "12345"
   - sbatch args: --job-name, --time, --cpus-per-task, --mem, --output, --error, --account
   - Wait loop checks squeue status with poll_interval
   - Terminal states: COMPLETED, FAILED, CANCELLED, TIMEOUT, NODE_FAIL, OUT_OF_MEMORY
   - If job not in queue and output file exists with errors, detect failure

4. **CLI interface:**
   - `python -m scripts.infra.executor detect`
   - `python -m scripts.infra.executor local --command "echo hello"`
   - `python -m scripts.infra.executor slurm --script run.sh --name test --time 01:00:00`
   - `python -m scripts.infra.executor status --job-id 12345`

5. **Error handling:**
   - Raise RuntimeError if sbatch fails
   - Return 'UNKNOWN' status if squeue fails
   - Handle missing output files gracefully

DO NOT:
- Use job arrays (independent jobs only per context decision)
- Block indefinitely on wait (always poll with interval)
- Assume specific Slurm configuration (check for sbatch command availability)
- Require account parameter (use None as default, omit from command)
