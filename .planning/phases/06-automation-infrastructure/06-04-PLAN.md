Create `scripts/infra/plugins/group_id_check.py` to validate MM/PBSA group IDs.

Function `validate_group_ids(workspace: Path) -> HandoffRecord`:
1. Locate files:
   - index_file = workspace / "index.ndx"
   - groups_file = workspace / "mmpbsa_groups.dat"

2. If index.ndx missing: return FAILURE with error "index.ndx not found - run trj4mmpbsa first"

3. Parse index.ndx to extract group names and IDs:
   - Pattern: `[ group_name ]` on line N means group_name has ID N (approximately)
   - Build dict: {group_name: group_id}

4. If mmpbsa_groups.dat exists:
   - Parse receptor/ligand group names (lines 2-3)
   - Validate group names exist in index.ndx
   - Emit WARNING if group name not found in index
   - Store validated IDs in data["receptor_group_id"], data["ligand_group_id"]

5. If mmpbsa_groups.dat missing:
   - Emit INFO: "mmpbsa_groups.dat not found - will be generated from index.ndx"
   - Attempt to auto-detect receptor/ligand groups (look for "Protein", "Receptor", "LIG", "Ligand")
   - Store suggested IDs in recommendations

6. Return HandoffRecord:
   - status: FAILURE if critical errors, NEEDS_REVIEW if warnings, SUCCESS if all valid
   - data: {receptor_group_id, ligand_group_id, validated: true/false}
   - errors/warnings/recommendations as appropriate

main() function:
- argparse: --workspace (required)
- Call validate_group_ids(Path(workspace))
- Print record.to_dict() as JSON
- Exit 0 if SUCCESS/NEEDS_REVIEW, 1 if FAILURE

Docstring: "Group ID consistency checker - validates MM/PBSA group IDs against index.ndx."

Create `scripts/infra/plugins/conversion_cache.py` following "Per-Workspace Cache Isolation" pattern from 06-RESEARCH.md.

ConversionCache class:
1. `__init__(self, workspace: Path, cache_type: str = "conversions")`
   - Set cache_dir = workspace / ".cache" / cache_type
   - Create cache_dir with parents=True, exist_ok=True

2. `cache_key(self, source_path: Path, target_format: str) -> str`
   - Generate deterministic key: sha256(f"{source_path.resolve()}:{target_format}")
   - Return hex digest

3. `get(self, source_path: Path, target_format: str) -> Path | None`
   - Compute cache_key
   - Check if cache_file and meta_file exist
   - Parse meta_file JSON
   - Compare source_mtime in meta vs current source_path.stat().st_mtime
   - If cache older than source: return None (stale)
   - If fresh: return cache_file path

4. `put(self, source_path: Path, target_format: str, result_path: Path)`
   - Compute cache_key
   - Copy result_path to cache_file
   - Write meta_file JSON: {source_path, source_mtime, target_format, cache_mtime}

5. `clear(self, source_path: Path = None)`
   - If source_path: remove cache entries for that source
   - If None: remove all cache entries in cache_dir

Do NOT implement main() - this is a library class, not a standalone plugin.
Add module docstring: "Conversion cache manager for deterministic file transformations."

Follow code example exactly from 06-RESEARCH.md "Pattern 3: Per-Workspace Cache Isolation".
