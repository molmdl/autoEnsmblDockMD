Implementation approach: Add a simple loop before `ensure_env` that checks for `--workdir` and shifts:
```bash
CUSTOM_WORKDIR=""
while [[ $# -gt 0 ]]; do
  case "$1" in
    --workdir) CUSTOM_WORKDIR="$2"; shift 2 ;;
    *) shift ;;
  esac
done
if [[ -n "$CUSTOM_WORKDIR" ]]; then
  cd "$CUSTOM_WORKDIR" || { echo "Error: cannot cd to $CUSTOM_WORKDIR" >&2; exit 1; }
fi
```
Place this BEFORE `ensure_env` and `find_workspace_root` so the workspace detection uses the specified directory.
Note: Remove `parse_flags "$@"` call since we're manually parsing and common.sh parse_flags may not know --workdir.
