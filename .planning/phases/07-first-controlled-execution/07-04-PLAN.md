  <action>
Execute MD equilibration and production runs:

1. Run MD production stages:
   ```bash
   source scripts/setenv.sh
   
   # Stage 4: MD production
   bash scripts/run_pipeline.sh --config work/test/config.ini --stage com_prod
   ```

2. Monitor Slurm job completion:
   ```bash
   # Production MD is asynchronous - check job status
   for LIGAND in dzp ibp; do
       if ls work/test/com/$LIGAND/*.job 2>/dev/null; then
           JOBID=$(basename work/test/com/$LIGAND/*.job .job | sed 's/.*_//')
           echo "Waiting for $LIGAND MD job: $JOBID"
           
           # Wait with timeout (24 hours max)
           TIMEOUT=86400
           ELAPSED=0
           
           while squeue -j $JOBID >/dev/null 2>&1 && [ $ELAPSED -lt $TIMEOUT ]; do
               echo "Job $JOBID still running... (${ELAPSED}s elapsed)"
               sleep 300
               ELAPSED=$((ELAPSED + 300))
           done
           
           # Check completion
           if [ $ELAPSED -ge $TIMEOUT ]; then
               echo "WARNING: Job $JOBID timeout after ${TIMEOUT}s"
           else
               sacct -j $JOBID --format=JobID,State,Elapsed,ExitCode
           fi
       fi
   done
   ```

3. Verify MD outputs:
   ```bash
   for LIGAND in dzp ibp; do
       echo "Checking $LIGAND MD outputs..."
       
       # Check trajectory
       ls work/test/com/$LIGAND/*.xtc 2>/dev/null || {
           echo "WARNING: No trajectory for $LIGAND"
       }
       
       # Check energy file
       ls work/test/com/$LIGAND/*.edr 2>/dev/null || {
           echo "WARNING: No energy file for $LIGAND"
       }
       
       # Check log
       ls work/test/com/$LIGAND/*.log 2>/dev/null || {
           echo "WARNING: No MD log for $LIGAND"
       }
   done
   ```

4. Check handoff status and log completion.

DO NOT proceed to Stage 5 if trajectory is missing - MM/PBSA requires trajectory.
DO monitor job completion with timeout to avoid infinite wait.
</action>
