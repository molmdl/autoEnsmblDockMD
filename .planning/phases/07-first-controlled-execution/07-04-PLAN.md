**Step 4: Verify handoff status**
```bash
status=$(jq -r '.status' .handoffs/docking_run.json)
echo "Handoff status: $status"

case "$status" in
    success)
        echo "✓ Skill executed successfully (job submitted)"
        ;;
    needs_review)
        echo "⚠ Warnings detected:"
        jq -r '.warnings[]?' .handoffs/docking_run.json
        ;;

        echo "✗ Skill execution failed with status: $status"
        jq -r '.errors[]?' .handoffs/docking_run.json

        echo "ERROR: Unknown status: $status"
        exit 1
        ;;
esac
```

**Step 3: Write checkpoint file**
```bash

Waiting for docking Slurm jobs to complete.

**Jobs:** $(echo "$job_ids" | tr '\n' ', ')
**Stage:** $stage
**Status:** $status

Check completion:

sacct -j $(echo "$job_ids" | tr '\n' ',') --format=JobID,State,Elapsed,ExitCode
\`\`\`

Expected outputs:
- dock/dzp/docked.sdf (dzp ligand docking)
- dock/ibp/docked.sdf (ibp ligand docking)
- dock/dzp/docking_scores.txt
- dock/ibp/docking_scores.txt

<next_action>
Resume when jobs complete:

1. Check job status:
   \`\`\`bash
   sacct -j $(echo "$job_ids" | tr '\n' ',')
   \`\`\`

2. Verify outputs exist

3. Load next skill:

   skill(name="aedmd-com-setup")

EOF
```
