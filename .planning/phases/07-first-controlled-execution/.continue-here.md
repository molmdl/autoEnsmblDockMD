**The phase's intent:**
Per 07-CONTEXT.md: "Test both human-facing tools AND agent skills for both docking modes"

**The discovery:**
Agent skills in `.opencode/skills/` are thin wrappers:
- `aedmd-rec-ensemble` → dispatches runner → calls `scripts/rec/0_prep.sh`
- `scripts/rec/0_prep.sh` → submits Slurm equilibration job → returns
- `scripts/rec/1_pr_rec.sh` → submits Slurm array job → returns
- Handoff JSON records success (submission OK) but job is still running

**The checkpoint flow needed:**
```
For each async stage:
1. Load skill via `skill` tool
2. Execute skill wrapper (submits Slurm job)
3. Parse job ID from handoff JSON
4. Write .continue-here.md with job ID
5. EXIT with "wait for job completion"
6. On resume: check sacct, verify outputs, proceed
```
