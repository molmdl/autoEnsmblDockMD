**scripts/agents/base.py:**
- Import AgentState from scripts.infra.state, CheckpointManager from scripts.infra.checkpoint
- `BaseAgent` abstract class (ABC) with:
  - `__init__(self, workspace: Path, config: Optional[Dict[str, Any]] = None)`: stores workspace (as Path), config (dict), creates AgentState(workspace / '.agent_state.json'), creates CheckpointManager(workspace / '.checkpoints')
  - `execute(self, input_data: Dict[str, Any]) -> Dict[str, Any]`: abstractmethod — takes handoff dict, returns handoff dict
  - `get_role(self) -> str`: abstractmethod — returns role identifier string
  - `save_checkpoint(self, stage: str, state: Dict[str, Any]) -> None`: delegates to self.checkpoint_mgr.save_checkpoint(stage, state)
  - `load_checkpoint(self, stage: str) -> Optional[Dict[str, Any]]`: delegates to self.checkpoint_mgr.load_checkpoint(stage=stage)
  - `create_handoff(self, to_agent, stage, status, data=None, warnings=None, errors=None, recommendations=None) -> Dict[str, Any]`: convenience method that creates HandoffRecord with from_agent=self.get_role(), converts to dict
  - Add logging via `logging.getLogger(f'agent.{self.get_role()}')` stored as self.logger

**scripts/agents/utils/routing.py:**
- `STAGE_AGENT_MAP`: dict mapping WorkflowStage values to agent role strings. All execution stages map to "runner", COM_ANA maps to "analyzer". INIT and COMPLETE map to "orchestrator".
- `get_agent_for_stage(stage: WorkflowStage) -> str`: returns the agent role for a given stage
- `get_stages_for_agent(role: str) -> List[WorkflowStage]`: returns list of stages handled by a given agent role

**scripts/agents/utils/__init__.py:** Export get_agent_for_stage, get_stages_for_agent.
