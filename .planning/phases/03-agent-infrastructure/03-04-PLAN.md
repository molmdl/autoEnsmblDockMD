  - From .checker: CheckerAgent

  - From .schemas: HandoffRecord, HandoffStatus, WorkflowStage
- Define `AGENT_REGISTRY: Dict[str, Type[BaseAgent]]` mapping role strings to classes:
  - "orchestrator": OrchestratorAgent
  - "runner": RunnerAgent
  - "analyzer": AnalyzerAgent

  - "debugger": DebuggerAgent
- Define `get_agent(role: str, workspace: Path, config: Dict = None) -> BaseAgent` factory function
- Add `__all__` list

**scripts/agents/__main__.py:**
- argparse CLI with arguments:
  - `--agent` (required): Agent type (choices from AGENT_REGISTRY keys)
  - `--workspace` (required): Workspace directory path
  - `--config` (optional): Path to config.ini file
  - `--input` (optional): Path to input handoff JSON file
  - `--output` (optional): Path to write output handoff JSON file (default: stdout)
- Main flow:
  1. Parse args
  2. Load config from --config if provided (use ConfigManager from scripts.infra.config)
  3. Load input handoff from --input JSON file if provided, else use empty dict
  4. Instantiate agent via get_agent()
  5. Call agent.execute(input_data)
  6. Write result to --output file or print to stdout as JSON
- Wrap in try/except, print errors to stderr, exit with appropriate code
- Include `if __name__ == '__main__':` guard
